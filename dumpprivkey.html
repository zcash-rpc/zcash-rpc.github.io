dumpprivkey - Zcash 4.7.0 RPC

dumpprivkey "t-addr"

Reveals the private key corresponding to 't-addr'.
Then the importprivkey can be used with this output

Arguments:
1. "t-addr"   (string, required) The transparent address for the private key

Result:
"key"         (string) The private key

Examples:
> zcash-cli dumpprivkey "myaddress"
> zcash-cli importprivkey "mykey"
> curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "dumpprivkey", "params": ["myaddress"] }' -H 'content-type: text/plain;' http://127.0.0.1:8232/
